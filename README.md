## API Reference

#### Get all Users

```http
  GET /api/users
```

#### Create a new user

```http
  POST /api/users
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `body` | `raw` | **Required**. all fields |


#### Update User

```http
  GET /api/users/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `body` | `raw` | **Required**. Id of user to update |

```http
  GET /api/users/${id}
```

#### Delete User

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of user to delete |
